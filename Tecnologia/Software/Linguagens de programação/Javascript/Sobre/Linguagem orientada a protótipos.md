- Conceito de linguagem orientada a protótipos é um paradigma de programação onde a herança e a criação de objetos são feitas através de protótipos, ou seja, outros objetos existentes, em vez de classes. Em vez de definir uma classe para depois criar instâncias, você clona um objeto existente (o protótipo) e estende suas propriedades e métodos para criar novos objetos.
