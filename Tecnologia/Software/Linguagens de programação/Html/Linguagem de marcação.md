- Uma linguagem de marcação é um sistema para definir a estrutura, a apresentação e/ou a finalidade do conteúdo dentro de um documento digital, para que os programas de computador possam interpretar e exibir o conteúdo corretamente.

- Linguagens de marcação codificam texto usando vocabulário, símbolos e/ou regras de sintaxe específicos.

- Por exemplo, a [tag <h1>](https://www.semrush.com/blog/h1-tag/) em HTML (linguagem de marcação de hipertexto) comunica que o texto contido nela é um título de nível superior.

- Os mecanismos de busca podem usar essas informações para entender melhor o assunto da sua página. Já os navegadores podem usar essas informações para aplicar o estilo apropriado.

- Um mecanismos de busca mais famoso é o Google.
