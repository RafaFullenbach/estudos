## Acessibilidade

- Uma técnica de navegação comum para usuários de softwares de leitura de tela é pular rapidamente de um título para outro a fim de determinar o conteúdo da página. Por isso, é importante não pular um ou mais níveis de título. Fazer isso pode gerar confusão, pois a pessoa que navega dessa forma pode ficar sem saber onde está o título que falta.
